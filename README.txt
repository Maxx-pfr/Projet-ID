Avant toute chose il est important de lire le manuel d'utilisation de l'application qui se trouve dans le dossier Rapport.
La note développeur se trouve dans le dossier Rapport.

L'éxecutable de l'application se trouve dans le dossier API Feet et s'appelle "API Feet.exe"

Notes importantes :

	- Désactiver le bluetooth de votre ordinateur avant de lancer l'application, en effet un problème de port COM peut se générer. Lors de la connexion de la carte Arduino, celle-ci se connecte sur le premier port que l'application détecte. Il est donc préférable de n'avoir aucun autre port COM d'activé.
